# Compiler


## [GrammarParserTools](./Tools/)

&#160;&#160;&#160;&#160; 文法检查工具：判断输入文法是否满足指定的编译解析文法类型。

&#160;&#160;&#160;&#160; 目前支持检查的文法类型包括`SLR(1)`和`LR(1)`。
